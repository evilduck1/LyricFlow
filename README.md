# LyricFlow

Tauri + Vite + React MP3 player with synced scrolling .lrc lyrics.

## Features
- MP3 playback
- Synced scrolling lyrics (.lrc)
- Resizable lyrics + playlist panels
- Internal scrolling only
- Playlist search + “Now Playing” scroll
- macOS-ready icon

## Dev
npm install
npm run tauri dev

## Build
npm run tauri build
