# LyricFlow

Tauri + Vite + React MP3 player with synced scrolling .lrc lyrics.

## Features
- MP3 playback
- Synced scrolling lyrics (.lrc) capability
- Playlist search + “Now Playing” scroll

## HOW TO RUN FOR MAC AND WINDOWS USRES
FOR MAC USERS ONLY ONCE MOVED TO THE APPLICATIONS FOLDER BUT BEFORE FIRST USE, RUN COMMAND IN TERMINAL xattr -dr com.apple.quarantine "/Applications/Prompt Saver.app" THIS IS A ONE TIME OPERATION
    
Windows SmartScreen Warning (First Launch)
    On the Windows protected your PC screen, click More info
    Click Run anyway
    The app will start normally


## Dev
npm install
npm run tauri dev

## Build
npm run tauri build
